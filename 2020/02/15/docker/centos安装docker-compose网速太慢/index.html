centos安装docker-compose网速太慢

作者: IFreeOvO 日期: 二月 15, 2020  14:28:38 分类: docker

问题描述

使用sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose安装docker-compose,下载速度几乎为0

解决方案

换个下载地址尝试sudo curl -L "https://get.daocloud.io/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
